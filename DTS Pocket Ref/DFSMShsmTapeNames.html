FDRABR

TAPE DATASET NAMING CONVENTION

Incremental and Full Volume Backup
     FDRABR.Vvolser.Cnggggcc
          n= copy number (1-9)
          gggg=generation number (1-9999)
          cc=cycle number (1-63)

Archive Backup

     FDRABR.Vvolser.bnyydddx
          b=identifier (B,D-J)
          n=copy number (1-2)
          yyddd = Julian date
          x = occurrence number (A-Z, 0-9)

Application Backup

     abrindex.Vvolser.bnyydddx
          abrindex = HLQ of Application A.C.F
          remainder = same as Archive Backup
